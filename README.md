# Basketball Points Counter

Basketball Points Counter is a modern scoreboard web application for tracking points during a basketball game.

The project was created as a JavaScript practice app focused on DOM manipulation, event handling, game logic and interactive UI elements.

Live demo: https://graceful-souffle-91128e.netlify.app/

## About the project

The application allows the user to control a simple basketball scoreboard.

It includes two teams, a game timer, period control, foul counters and a new game button. The interface is inspired by a digital sports scoreboard with a dark background and bright display-style numbers.

The app can be used to simulate score tracking during a basketball match or as a coding project for practicing JavaScript logic.

## Features

- Adding points for Team 1,
- Adding points for Team 2,
- Adding 1, 2 or 3 points,
- Tracking fouls for both teams,
- Increasing and decreasing fouls,
- Period control,
- Game timer,
- Start, pause and reset timer buttons,
- New game button,
- Leader status display,
- Digital scoreboard-style UI,
- Live version deployed on Netlify.

## How to run locally

1. Clone the repository:

```bash
git clone https://github.com/laskowskialeks/Basketball-Points-Counter.git
```

2. Open the project folder:

```bash
cd Basketball-Points-Counter
```

3. Open the `index.html` file in your browser.
